### Install Node.js 22
Download and install from: https://nodejs.org/

### Install Angular CLI 22
```bash
npm install -g @angular/cli
```

### Verify Installation
```bash
node --version
ng version
```

### Build
```bash
ng build
```

## Configuration

Update the following files in the `public/` directory:
- **CV/Resume**: Add your resume file
- **Profile Picture**: Add your profile picture
